## Codebase Navigation

This repository uses `AGENTS.local.md` files for AI-readable directory summaries.
These files are git-excluded (local only) and generated by the codebase-navigator agent.

**When navigating the codebase:**
- Before globbing or reading source files in a directory, check for `AGENTS.local.md` there first.
- Trust the summary for orientation. Read source only when the summary is insufficient.
- If `AGENTS.local.md` is missing for a directory you need to understand, say so — the user
  can regenerate it with: `@codebase-navigator update this directory`.

**When `OUTDATED.local.md` exists at the repo root:**
- Flag it to the user at the start of the session.
- Offer to process it: `@codebase-navigator process outdated`.

**Do not commit `AGENTS.local.md` or `OUTDATED.local.md`.** They are machine-local.
